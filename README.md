## Multiselect Dropdown for Flutter.

A customizable, lightweight multiselect dropdown package for Flutter.

## Features

✅ Checkbox-style multiselect  
✅ Custom label mapping  
✅ Easily embeddable in forms  
✅ Works with any data model
